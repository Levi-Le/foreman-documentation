= Importing an Ansible Playbook by Name

You can import Ansible playbooks by name to {Project} from collections installed on {SmartProxy}.

.Procedure
. Fetch the available Ansible playbooks using the following API request:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
curl -X GET 'Content-Type: application/json' https://_{foreman-example-com}_:{smartproxy_port}/ansible/api/v2/ansible_playbooks/fetch?proxy_id=_proxy_id_number_ | json_pp
----
. Select the Ansible playbook you want to import and note its name.
. Import the Ansible playbook using its name:
+
[options="nowrap", subs="+quotes,verbatim,attributes"]
----
curl -X PUT 'Content-Type: application/json' -d '{ "playbook_names": ["_playbook_name_"] }' https://_{foreman-example-com}_:{smartproxy_port}/ansible/api/v2/ansible_playbooks/sync?proxy_id=_proxy_id_number_ | json_pp
----
You get a notification in the {ProjectWebUI} once the import completes.
